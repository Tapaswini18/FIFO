# FIFO
Implementation of FIFO (First In First Out) data structure where the first element inserted is the first one to be removed. Demonstrates queue operations such as enqueue and dequeue using programming logic.
## Features
- Enqueue operation (insert element)
- Dequeue operation (remove element)
- Display queue elements

## Applications
- CPU scheduling
- Data buffering
- Printer queue management

## Technologies Used
- C / Python
